Green
Journey end / clock-out / logout
Last recorded event of the day. X icon = "this is where the work stopped". Used for CLOCK_OUT and LOGOUT activities.

Meeting started
Agent crossed the 50m proximity threshold and started a meeting. Rectangle icon = "a session is open". Maps to MEETING_START.

Visual grammar — how to read the map at a glance
Blue path line
The route taken. No markers needed along it — the line is the story.
Pulsing ring
Only on live/cyan marker. Pulse = alive right now. Everything else is history.
Warm → cool
Green = good/start. Red = stop/urgent. Cyan = now. Violet = person. The hue tells you the mood before you read the label.
Icon inside pin
Circle = start. X = end. Rectangle = meeting. Silhouette = person. You can identify the event without reading any text.
